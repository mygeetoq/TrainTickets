# Интеллектуальный чат-бот для продажи ЖД билетов

Этот проект представляет собой курсовую работу по предмету «Интеллектуальные системы и технологии» (Вариант 4). Бот разработан на Python с использованием методов машинного обучения и обработки естественного языка (NLP).

## Основные возможности
1.  **Классификация намерений (ML)**: Используется `LinearSVC` и `TfidfVectorizer` для понимания запросов пользователя (приветствие, покупка билета, цена, расписание).
2.  **База диалогов**: Поиск ответов в датасете `dialogues.txt` с использованием расстояния Левенштейна (NLTK `edit_distance`).
3.  **Рекламный модуль**: Ненавязчивое предложение дополнительных услуг (страховка, питание, бизнес-зал).
4.  **Интеграция с Telegram**: Полноценный интерфейс мессенджера.

## Структура файлов
- `bot_core.py`: Ядро бота (логика NLP, ML-модель, обработка диалогов).
- `telegram_bot.py`: Оболочка для работы через Telegram API.
- `intents.json`: Расширенный список намерений и паттернов.
- `dialogues.txt`: Датасет типовых диалогов.
- `products.json`: Список рекламных товаров.
